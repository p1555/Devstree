Summary & Server-side Alternatives

  • Geolocation: Use forms to let users paste coordinates, or obtain approximate location server-side from IP (low accuracy).
  • Drag & Drop: Use file upload forms or server-side widgets.
  • Web Storage: Persist on server via sessions/database or set cookies from server responses.
  • Canvas: Serve pre-rendered images or SVGs; accept image uploads from users.
  • Audio/Video: Works natively with <audio> and <video> tags.
